# MoodTune - Your Personal Music & Mood Journal

MoodTune is an engaging web app that helps users track their daily mood alongside the music they listen to. Users can:
- Log their daily mood using an intuitive emoji slider
- Add the song they're currently listening to
- See beautiful visualizations of their mood patterns
- Get personalized music recommendations based on mood history
- Share their mood-music combinations with friends

Built with vanilla JavaScript and CSS animations for a smooth, engaging experience.
